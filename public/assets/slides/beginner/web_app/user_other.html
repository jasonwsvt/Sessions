Web App - User Login and New User Functionality

Log in as a different local user

Click the button to go to the login menu.

Every local account that's not set as hidden in the settings menu and isn't the currently logged in account will show up here.

For any hidden account, just type it in to the field provided.

Any account without a password will automatically log in.

If the account has a password, you'll need to provide it before logging in.


New User creation

Click the button to create a new account called "newuser" and open the settings menu.

If there is currently an account with a username of "newuser", the button is disabled.

Once there is no account with "newuser" as the username, the button will re-enable.
