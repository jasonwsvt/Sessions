Web App - Component Buttons

In the bottom left of the window is a list of button groups, and in the bottom right is a list of component buttons, grouped in the same order as the list on the left.

Clicking a group button will make the selected group of component buttons visible and scroll to the selected group, and make it visible if it isn't.

Ctrl+clicking a group button will toggle hidden the selected group of component buttons.

Shift+clicking a group button will toggle hidden all other groups of component buttons.

Clicking one of the component buttons on the right will add the component to the session editor at the current cursor position.
