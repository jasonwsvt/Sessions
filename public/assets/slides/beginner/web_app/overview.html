Web App - Overview

The app has three areas, in order from top to bottom - the utility toolbar, the session editor, and the component buttons.

Spanning the top of the window is the utility toolbar. It consists of user, client, issue, and session utilities.

Far to the right of the toolbar are buttons to adjust the size of the session editor and component buttons sections, and open the instructions.
No further explanation for these buttons is needed.

I've tried to design this web app to be fairly robust and straight-forward in operation.

You can use the app without creating an account.

It won't back up anything you've done if you don't set the backup frequency in user settings.

A lot of effort went into ensuring functionality is disabled when using it would cause problems.
For the most part it works well.

There are some bugs I'm aware of, however, and there are likely more I'm not.

My hope is there will be enough interest in the app and method to put further effort into its improvement.

Keep in mind, this is my first web project, and it's been a solo endeavor from the beginning.
If you have suggestions for improvement, I welcome them.
