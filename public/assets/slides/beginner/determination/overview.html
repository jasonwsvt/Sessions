Determination is the process of using energy testing to find all the reasons that underlie an issue.

Each reason is a set of connected components for an issue, and one important way they're connected is that only when determined sufficiently can the reason be completely released.

On the next several slides I will detail the more tangible components that are often contained within reasons: the physical body, senses, emotions, psychological reversals, and patterns.

I will also cover a simple method to locate items in lists using energy testing - Binary Search.
