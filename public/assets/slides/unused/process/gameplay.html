Process - Gameplay

  1. Write down a symptom or aspiration.
    If you like, you could figure one out using energy testing.
  2. Energy test if there’s a reason.
    You could test the number of remaining reasons but it might change.
  3. Determine the number of components for the reason.
  4. For each component, determine the type (psychological reversal, emotion, event aspect (when, where, who, what, why, how, etc.), physical body, sense, energy system, low level energy, characteristic (of others, extranormal level, negation, etc.), or something else).
  5. Once the reason’s components are determined, resolve the reason.
    It might need to be put in the form of a statement.
  6. If a reason doesn’t resolve, it has one or more underlying reasons.
    Indent and go back to step 2 to determine them!
  7. If the reason does resolve, go back to step 2 to see if there’s another.
  8. When you clear the original issue you’re done with the round!
    If you like, start another round by going back to step 1!
