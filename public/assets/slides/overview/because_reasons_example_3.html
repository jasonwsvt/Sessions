So what does this actually look like?

A quick example:

Issue: I don’t like creating this presentation. It’s a whole lot of work and it might end up being a waste of time.

Reasons (3):

  1. Deprivation (I feel deprived by the amount of time it's taking, which I could be using for other things.)
  2. Deep level recurrent wariness (At a very deep level, I question how necessary it is that I need write this presentation.)
  3. I’m worried no one will find interest in it.

Now I’m checking to see if the original issue is still true. It’s not.

Is the inverse true? I do like creating this presentation. It’s not too much work and it might end up being worth it. Yes, that’s true.
