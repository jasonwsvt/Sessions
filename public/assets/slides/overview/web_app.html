Overview of this web app

Below is an image and chart showing all the different parts of the web app.

SectionTitleDescription
A Utility Toolbar
User Utility
  1. View or edit settings for the current user
  2. View or modify records for the current user
  3. Log in to a different account
  4. Create a new user account
 Client Utility
  1. View all the clients for the current user
  2. Rename the currently selected client
  3. Add a new client for the current user
 Issue Utility
  1. View all the issues for the current client
  2. Rename the currently selected issue
  3. Add a new issue for the current client
 Session Utility
  1. View all the sessions for the current issue
  2. Add a new session for the current issue
 Etc. Section
  1. Make the height of the current session textarea smaller
  2. Make the height of the current session textarea larger
  3. Open the information utility
B Current Session Textarea This area is the editor for the currently selected session.
Acceptable keys are any letter, number, symbol, space, and enter. Use tab and shift-tab to adjust indentation for the current line.
C Component Buttons On the left is a list of component group buttons. Clicking one will scroll the component list on the right to the selected group. On the right is a long list of components. Clicking one of these buttons will add the component in the current session textarea at the current cursor position.

The Beginner level has more detailed instructions for the app.
