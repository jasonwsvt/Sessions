Overview of this web app

Below is an image and chart showing all the different parts of the web app.

SectionTitleDescription
A Utility Toolbar
User Utility
  1. View or edit settings
  2. View or modify records
  3. Log in to another account
  4. Create a new user account
 Client Utility
  1. View all the clients
  2. Rename the currently selected client
  3. Add a new client
 Issue Utility
  1. View all the issues
  2. Rename the currently selected issue
  3. Add a new issue
 Session Utility
  1. View all the sessions
  2. Add a new session for the current issue
 Etc. Section
  1. Make the height of the current session textarea smaller (Ctrl+Up)
  2. Make the height of the current session textarea larger (Ctrl+Down)
  3. Open the information utility
B Current Session Textarea This area is the editor for the currently selected session.
Any typed letter, number, symbol, space, and enter will add to the session at the cursor location.
Use tab and shift-tab to adjust indentation for the current line.
Move the cursor by pressing up, down, left, and right.
C Component Buttons On the left is a list of component group buttons.
Click: Make the selected group visible.
Ctrl+Click: Toggle hidden the selected group.
Shift+Click: Toggle hidden all other groups.
If the selected group becomes visible, scroll to it. 		On the right is a long list of components. Clicking one of these buttons will add the component to the current session textarea at the current cursor position.

The Beginner level has more detailed instructions for the app.
