Numbers - Determining the number of places in very big or small number:

Previously we learned how to determine a number under a million or so. What about an even larger number, or a number between 0 and 1? For these I determine the precision, then use scientific notation, where the range would be X*10^N <= number < (X+1)*10^N. Precision is a fancy word for the number of decimal places necessary for required accuracy. For more about scientific notation, check out this link: https://www.mathsisfun.com/numbers/scientific-notation.html

Here's the general process I use:

1) Determine the values for N, then X, then the sufficient precision.

2) Use the binary search method to determine the number.
