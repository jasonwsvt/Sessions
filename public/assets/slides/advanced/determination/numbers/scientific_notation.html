Numbers - Scientific Notation:

Very large and very small numbers are often easier to display in scientific notation.

Scientific notation uses the following format: M x 10P, where M is the mantissa, and P is the power.

Using this way of writing numbers, any number will be multiplied by P powers of ten so that M is between 0 and 10, but not equal to 0 or 10.

Here are some examples:

Actual numberNumber shown in scientific notation
00
.0011 x 10-3
.022 x 10-2
.33 x 10-1
44 x 100
505 x 101
6006 x 102
34592250103.45922501 x 109
.000000000000099999.999 x 10-14

Note how the range of the mantissa by itself is always between 0 and 10, and the power is the number of places the decimal point is moved (positive numbers indicate leftward movement, and negative numbers rightward movement).
