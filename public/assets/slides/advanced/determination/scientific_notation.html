Numbers - Scientific Notation:

Very large and very small numbers are often easier to display in scientific notation. Using this way of writing numbers, any number will be multiplied by a power of ten so that it is in the range 0 < number < 10. You can think of N as the number of places the decimal point is moved to the left (negative numbers indicate backward movement, or to the right). Here are some examples:
