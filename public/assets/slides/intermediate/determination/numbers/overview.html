Issue Determination - Numbers:

Numbers come up fairly frequently in energy testing -

  • consciousness levels,
  • number of psychological reversals,
  • number of issues,
  • number of reasons,
  • number of components in an issue or reason,
  • number of a single component, or
  • number of a single component type.

I have a few methods for quickly determining numbers. I first determine the number of places, then I determine the number for each place. On the next slide I'll describe a quick method I use for determining a number between 0 and 9, and on the following slide I'll detail the process I use for determining the number of places.
