Numbers - Determining the value of a one digit number:

To determine a number between 0 and 9, I find it helpful to imagine the numbers in this format:

Low0123
Medium456
High789

I then use a modified binary search to find the number by doing the following:

1) I energy test whether the number is low, medium, or high.

2) I then energy test which number it is in that group.

For an example, I'll search for the number 8:

TestResponse
The number is in the low group.False.
The number is in the medium group.False.
By elimination it is known the number is in the high group. The number is 7.False.
The number is 8.True.
