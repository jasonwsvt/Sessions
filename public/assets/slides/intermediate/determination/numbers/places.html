Numbers - Determining the number of places in a number:

To find a number in a range we use the binary search method as I described in the previous level.

But if the maximum point in the range, or the number itself, is exceptionally large, a method is needed to find it.

To determine a number between 0 and 9, I do the following:

1) I energy test whether the number is below 5.

1A) If true, I energy test if the number is 0, 1, 2 or 3.

2) If false, I energy test whether the number is above 5.

2A) If true, I energy test if the number is 7, 8, or 9.

3) If false, I energy test if the number is 4, 5, or 6.
