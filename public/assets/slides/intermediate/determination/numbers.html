Numbers - Between One and a Million or so:

When I need to find a number and the range of that number is unknown, here's the general process I use:

  1. Determine the precision.
  2. From the following list, determine the minimum number of the range:
    one, ten, one hundred, one thousand, one million, etc.
  3. The maximum number for the range is just below the following entry.
  4. Use binary search to find the number.
